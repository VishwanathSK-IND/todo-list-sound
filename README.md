# Ding! To-Do List

A simple, interactive To-Do List app that plays a satisfying "Ding!" sound when you complete a task.

## Features
* **Add Tasks:** Type a task and hit Enter or click Add.
* **Audio Feedback:** Uses the JavaScript Audio API to play a sound on completion.
* **Persistent Data:** Saves your list to LocalStorage so you don't lose tasks when you refresh.
* **Responsive:** Looks good on mobile and desktop.
